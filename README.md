CollegeHub

Overview
A full-stack college discovery platform where users can browse, filter, compare and save colleges.

Features
- College Listing
- Search
- State Filter
- Course Filter
- Rating Filter
- College Details
- College Comparison
- Authentication
- Saved Colleges

Tech Stack
- Next.js
- TypeScript
- TailwindCSS
- PostgreSQL
- Prisma
- Neon
- Vercel

Setup Instructions

1. Clone repository
2. Install dependencies
3. Configure environment variables
4. Run development server

Environment Variables

DATABASE_URL=
JWT_SECRET=
NEXT_PUBLIC_APP_URL=

Deployment
Hosted on Vercel
